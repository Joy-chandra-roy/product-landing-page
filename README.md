# JoySound – Wireless Headphone Landing Page

A modern and responsive landing page for a wireless headphone brand, built with React and Tailwind CSS. This project focuses on clean UI design, smooth animations, and high conversion layout.

## Live Demo
[view live project](https://product-landing-page-woad-sigma.vercel.app/)

## Screenshot
![project screenshot](/src/assets/product-landing-page.png)

## Features
--Beautiful Hero Section with CTA
--Fully Responsive Design (Mobile, Tablet, Desktop)
--Smooth Animations using Framer Motion
--Clean Navigation with Scroll
--Testimonials Section
-- Pricing Section
--FAQ Section
--Modern UI (Gradient, Glassmorphism, Shadow)

## Tech Stack
- Vite
- React.js
- Tailwind css
- Framer motion
- Swiper
- Lucide-react
- React icons
